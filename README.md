# 🏛️ Homebrew Formulae — Sirsi Tools

Official Homebrew tap for [Sirsi Technologies](https://github.com/SirsiMaster) tools.

## Installation

```bash
brew tap SirsiMaster/tools
brew install sirsi-pantheon
```
